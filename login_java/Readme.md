# How to Connect to MySQL/postgreSql Database, Insert, Update and Delete in Java NetBeans

Create database "connector" and table "connector"
ID, StudentID, Firstnme, Surname, Address, PostCode, Telephone

Open NetBeans --> New project --> Java eith Ant --> Java Application -->
        Project Name: Java_MysqlConn (uncheck the Create Main Class)

==> Select Java_MysqlConn --> New --> JFrame Form -->  
		ClassName: Java_MysqlConn
		Package: Java_MysqlConn
		
==> Rigth click on frame --> SetLayout -->Absolute Layout
==> Rigth click on frame --> Properties --> code tab --> Designer Size (1000, 800) and select Generate Center.
